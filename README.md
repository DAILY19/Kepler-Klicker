# Kepler Klicker

A space-themed incremental clicker game. Mine stardust by clicking planets, buy upgrades to automate production, and watch your planet evolve as you progress.

## Features

- **Click to mine** — Tap/click the planet to earn stardust
- **7 upgrades** — Mining Probes, Harvester Drones, Orbital Stations, Ice Colonies, Lava Forges, Terraformers, Singularity Engines
- **Planet evolution** — Planet appearance changes at milestones
- **Offline earnings** — Earn stardust while away (50% efficiency, 8hr cap)
- **Auto-save** — Progress saved to localStorage every 30 seconds
- **Background music** — Sci-fi ambient loops
- **Mobile-first** — Works on phones and desktop

## Running Locally

### Quick Start (Windows)

```powershell
.\start.ps1
```

### Manual (any OS)

```bash
npm install
npx http-server . -p 8080 -c-1 --cors
```
